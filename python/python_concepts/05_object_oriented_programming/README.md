# 05 Object Oriented Programming

Classes, inheritance, polymorphism, and OOP concepts

## Files in this folder:

### Basic OOP Concepts
- `oop_class_basics.py` - Basic class creation and instantiation
- `basic_class_example.py` - Simple class example
- `oop_encapsulation_demo.py` - Encapsulation demonstration

### Inheritance
- `inheritance_example.py` - Basic inheritance example
- `multiple_inheritance_example.py` - Multiple inheritance demonstration

### Polymorphism
- `polymorphism_examples.py` - **Comprehensive polymorphism examples including:**
  - Method Overriding (Runtime Polymorphism)
  - Duck Typing
  - Operator Overloading
  - Method Overloading Simulation
  - Polymorphic Collections

### Advanced OOP Features
- `abstract_class_example.py` - Abstract base classes
- `abstract_class_simple_example.py` - Simple abstract class example
- `property_decorator_explanation.py` - Property decorator usage
- `property_simple_example.py` - Simple property example

### Access Control and Attributes
- `attribute_access_comparison.py` - Comparing different attribute access methods
- `person_private_attributes.py` - Private attributes example
- `person_protected_attributes.py` - Protected attributes example
- `private_vs_public_demo.py` - Private vs public attributes comparison

### Other OOP Concepts
- `constructor_destructor_demo.py` - Constructor and destructor examples
- `custom_exception_demo.py` - Custom exception classes
- `pass_by_reference_example.py` - Pass by reference demonstration
